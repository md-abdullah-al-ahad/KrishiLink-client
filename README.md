# KrishiLink - Agricultural Marketplace

**Live Site:** [https://krishilink-agri.netlify.app/](https://krishilink-agri.netlify.app/)

## About KrishiLink

KrishiLink is a modern agricultural marketplace platform that connects farmers directly with buyers, enabling seamless crop trading and fostering agricultural growth in Bangladesh.

## Key Features

- **Direct Farmer-Buyer Connection** - Farmers can list their crops with detailed information including price, quantity, location, and quality descriptions, while buyers can browse and express interest directly.

- **Smart Crop Marketplace** - Browse all available crops with advanced search and filtering capabilities, view detailed crop information with high-quality images, and make informed purchasing decisions.

- **Interest Management System** - Buyers can send interest requests for crops with custom quantities and messages, while sellers can review, accept, or reject incoming interests through a comprehensive dashboard.

- **User Profile & Authentication** - Secure Firebase authentication with Google Sign-In support, personalized user profiles with editable information, and role-based access for buyers and sellers.

- **Responsive Modern Design** - Beautiful, mobile-first interface with gradient accents and smooth animations, glassmorphism effects in navigation, and optimized experience across all device sizes.

- **Real-time Updates** - Track your posted crops with instant status updates, monitor received interests with detailed buyer information, and manage your marketplace activity efficiently.

## Technologies Used

- **Frontend Framework:** React.js 18+ with Hooks (useState, useEffect, useContext)
- **Routing:** React Router v6 for seamless navigation and protected routes
- **Authentication:** Firebase Authentication with email/password and Google OAuth
- **Styling:** TailwindCSS & DaisyUI for modern, responsive component design
- **Form Management:** React Hook Form for efficient form validation and handling
- **Backend API:** RESTful API with MongoDB for data persistence
- **State Management:** React Context API for global authentication state
- **Notifications:** React Hot Toast for elegant user feedback
- **Build Tool:** Vite for fast development and optimized production builds
- **Deployment:** Netlify for continuous deployment and hosting
